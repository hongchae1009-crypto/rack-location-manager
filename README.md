# Rack Location Manager

시약·샘플 바이알의 **물리적 랙 위치**를 랙 격자에서 클릭(또는 QR 스캔)만으로 기록하는 **오프라인 단일 HTML 툴**.
문장 타이핑 없이 슬롯을 클릭하면 위치코드가 자동 생성되고, 이 맵 자체가 "항상 그 자리" 규칙이 됩니다.

**라이브:** https://hongchae1009-crypto.github.io/rack-location-manager/

## 특징
- **랙 그리드 클릭 배정** → 위치코드 자동 생성 (예: `331-4-RACK-A-05`). 설정에서 형식 자유 편집.
- **랙별 행×열 자유 설정** (5×12, 3×7 등), 여러 랙 공존.
- **바이알 단위 배치** — 같은 코드라도 용량이 다른 바이알을 각각 다른 슬롯에.
- **QR/바코드**: USB 스캐너 · 카메라(QR) · 병/슬롯 QR 라벨 인쇄.
- **가져오기/내보내기**: 자신의 재고 `.xlsx`/CSV를 로컬에서 불러오기, `Pin,Location` CSV·전체 CSV·JSON 백업.

## 데이터/프라이버시
- 이 공개판에는 **시약 데이터가 포함돼 있지 않습니다.** 각자 `가져오기`로 본인 재고 파일을 불러오세요.
- 모든 데이터는 **브라우저 localStorage에만** 저장되고 서버로 전송되지 않습니다. (외부 CDN 3종은 라이브러리 로드용)

## 실행
- **기본:** `index.html` 더블클릭 (또는 위 라이브 링크). 그리드·USB 스캔·CSV/JSON·라벨 동작.
- **카메라 QR:** `https`(라이브 링크) 또는 `http://localhost`에서만. 로컬은 `python -m http.server 8000` 후 `http://localhost:8000`.

## 백업
작업한 배치는 브라우저 localStorage에 저장됩니다. **내보내기 → 전체 백업(JSON)** 으로 주기적으로 파일 백업을 권장합니다.
